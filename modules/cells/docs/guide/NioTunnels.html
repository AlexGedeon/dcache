Using NIO

Nio can only be enabled for LocationMgrTunnel(s) and only if no alternative SocketFactory is used. Nio Tunnel can talk to Non Nio tunnels.

Enabling NIO

Enabling NIO for all LoginManagers and/or LocationManager(s) within a domain. 
     set context niochannel true
Enabling NIO for a single LoginManager or LocationManager 
        -niochannel=true
